หลักการทำงานของป้ายนี้ (อธิบายอย่างง่าย)
1. กล้อง IP Camera จะตรวจจับรถและส่งไปยัง Raspberry Pi 3
2.Raspberry Pi 3 ที่ใช้หลักการเขียน Image processing เขียนมาร์ค เพื่อจับช่องของพิ้นที่จอดรถ
3.เมื่อ Raspberry Pi 3 ประมวลผล จะส่งค่าไปยัง บอร์ด Arduino Wemos D1
4. บอร์ด Wemos D1 จะส่งค่าไปยัง SEVEN SEGMENT
